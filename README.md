# Rule-Driven-Technical-Support-Case-Management-System

Overview

A CLI-based Technical Support Case Management System built using Python to simulate real-world L1/L2 support workflows.
The project focuses on structured troubleshooting, priority assignment, and ticket lifecycle management using rule-driven logic instead of hardcoded flows.
